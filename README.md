# 🍫 Chocolate VIP | High-End Experience

Interface imersiva desenvolvida para campanhas de Páscoa de alto padrão, focada em transmitir exclusividade e desejo através do design.

## 🚀 Engenharia e Estética
- **Visual Gastronômico:** Paleta em tons de cacau e dourado (`#d4af37`) para reforçar o valor premium.
- **Tipografia:** Uso da *Playfair Display* para conferir elegância e autoridade visual.
- **Background Imersivo:** Integração de vídeo em loop com filtros de brilho para profundidade de interface.
- **Performance:** Estrutura semântica leve com deploy acelerado.
